## 4. 回滚方法

若要恢复任一 `LOCAL_ONLY` 文件到 Git 原路径：

```bash
mv outputs/markdown_local_only_2026-04-21/<原路径> <原路径>
```

例如：

```bash
mv outputs/markdown_local_only_2026-04-21/docs/notes/minimal_api_patch_note.md docs/notes/minimal_api_patch_note.md
```

如果要整批恢复：

```bash
find outputs/markdown_local_only_2026-04-21 -name '*.md' -type f | while read -r f; do
  rel="${f#outputs/markdown_local_only_2026-04-21/}"
  mkdir -p "$(dirname "$rel")"
  mv "$f" "$rel"
done
```

恢复后再用 `git status --short` 确认删除状态消失。
